Decode a string from a specific encoding into generic b::string format.

Encodings are specified as a template parameter. Possible values: b::enc::utf8, b::enc::utf16, b::enc::utf32, b::enc::utf16_wide, b::enc::ascii, b::enc::iso8859_1, b::enc::cp1252 (Windows only), b::enc::os_native and b::enc::binary.

Warning
Be careful with the distinction between ASCII, cp1252 and ISO-8859-1 (which is Latin-1). Encodings are often mislabeled as ASCII when they are in fact something else.

Parameters

Returns

◆ encode() [1/2]

template<typename encoding >
auto b::string::encode ( ) const
inline

Encode this string into a fitting string type with a specified encoding.

Encodings are specified as a template parameter. Possible values: b::enc::utf8, b::enc::utf16, b::enc::utf32, b::enc::utf16_wide, b::enc::ascii, b::enc::iso8859_1, b::enc::cp1252 (Windows only), b::enc::os_native and b::enc::binary.

Warning
Be careful with the distinction between ASCII, cp1252 and ISO-8859-1 (which is Latin-1). Encodings are often mislabeled as ASCII when they are in fact something else.
Make sure to follow https://utf8everywhere.org/. Do not ever write non-UTF-8 to a file.
Template Parameters
encodingThe desired encoding of the output data
Parameters
strThe input string
Returns
The fitting string type for the encoding, such as std::string, std::u32string, etc...
