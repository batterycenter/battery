◆ sleep_precise()

void b::sleep_precise ( double  seconds)

Busy-wait for said seconds (highest possible accuracy).

Use this function only if you need very precise timing.

Warning
This function does not suspend the thread. Instead it busy-waits and thus causes full CPU usage while doing nothing.

[in]secondstime to sleep

◆ time()

double b::time ( )

Get the time the program has been running in seconds (since battery was initialized).

Can be used for all sorts of high-level timing and measurements. Units is seconds, but the precision is dependent on the platform, going down in the microseconds range.

Returns
How many seconds the program has been running
